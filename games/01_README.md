# Guess The Missing Number From Sequences Game

دي لعبة بسيطة C++ console game هدفها تدريبك على:
- التعامل مع Arrays و 2D Arrays  
- استخدام if statements للتحقق من الإجابات 
- التعامل مع user input/output بطريقة تفاعلية  

## Gameplay / طريقة اللعب
1. البرنامج هيعرض 5 number sequences فيها رقم ناقص (`?`)  
2. المستخدم يدخل الرقم الناقص لكل sequence  
3. البرنامج يحسب points بتاعتك من 5  
4. بعد كده يسألك لو عايز تشوف correct answers (y = نعم، n = لا)  
5. لو اخترت Yesا لإجابات الصحيحة هتظهر  

## Main Idea / الفكرة الرئيسية
- ممارسة logic thinking في حل المسائل العددية  
- التعلم العملي على arrays والتعامل مع conditions 
- تحسين مهارات C++ programming بطريقة سهلة وواضحة  

## Notes / ملاحظات
- لو المستخدم يدخل أي حرف غير `y` أو `n`, البرنامج هيبلغه إن الاختيار غير صالح  
- الكود مناسب جدًا للمبتدئين لتطبيق المفاهيم الأساسية
